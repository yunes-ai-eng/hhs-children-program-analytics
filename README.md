# HHS Children Program Analytics Dashboard

## Overview

The HHS Children Program Analytics Dashboard is an AI-powered data analytics platform developed to analyze, visualize, and forecast child program performance data from the U.S. Department of Health and Human Services (HHS).

The project combines interactive business intelligence dashboards, machine learning techniques, anomaly detection, and time-series forecasting to support data-driven decision making.


## Objectives

* Analyze historical program performance data.
* Identify trends and patterns in child program participation.
* Detect unusual or abnormal observations using anomaly detection techniques.
* Forecast future trends using machine learning forecasting models.
* Provide an interactive dashboard for stakeholders and decision makers.


## Key Features

### Interactive Analytics Dashboard

* Dynamic visualizations using Plotly.
* Real-time filtering and exploration of data.
* Clean and responsive Streamlit interface.

### Forecasting Module

* Time-series forecasting using Prophet.
* Future trend prediction.
* Confidence interval visualization.

### Anomaly Detection

* Automatic identification of unusual observations.
* Visual highlighting of anomalies.
* Support for exploratory data analysis.

### Data Exploration

* Summary statistics.
* Trend analysis.
* Program performance monitoring.
